4.2.- AzureML (Azure Machine Learning)

Azure Machine Learning es el servicio diseñado para crear y entrenar modelos de inteligencia artificial completamente desde cero

, con la ventaja de poder conectarlos fácilmente con otros servicios necesarios en tu proyecto de manera eficiente e integrada. Cuenta con un espacio de trabajo específico llamado Studio, donde puedes iniciar nuevos modelos, gestionarlos, monitorear su rendimiento mediante métricas y revisar el historial de todas las ejecuciones realizadas.

La interfaz de Azure ML Studio permite trabajar con cuadernos Jupyter Notebook (documentos interactivos que combinan código, visualizaciones y texto explicativo), utilizar bibliotecas especializadas para aprendizaje automático y tener la flexibilidad de ajustar los hiperparámetros (configuraciones que controlan cómo aprende el modelo) directamente en el código. Además, incluye una herramienta de depuración para encontrar y corregir errores, y permite la integración con Visual Studio Code (un popular entorno de programación), lo que facilita incorporarlo a proyectos que ya están funcionando en la nube.
