10. Resumen de Configuraciones Típicas

Clasificación Binaria

# Capa de salida
layers.Dense(1, activation='sigmoid')

# Compilación
model.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])

Clasificación Multiclase (N clases)

# Capa de salida
layers.Dense(N, activation='softmax')

# Con etiquetas one-hot
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# Con etiquetas como enteros
model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])

Regresión

# Capa de salida (sin activación o relu para valores positivos)
layers.Dense(1)  # o activation='relu' si solo valores positivos

# Compilación
model.compile(optimizer='adam', loss='mse', metrics=['mae'])

Resumen de Configuraciones por Tipo de Problema

Tipo de problema Capa de salida Loss Métricas sugeridas
Clasificación binaria Dense(1, activation='sigmoid') binary_crossentropy accuracy, precision, recall, AUC
Clasificación multiclase (one-hot) Dense(N, activation='softmax') categorical_crossentropy accuracy, top_k_accuracy
Clasificación multiclase (enteros) Dense(N, activation='softmax') sparse_categorical_crossentropy accuracy, top_k_accuracy
Regresión (valores cualquiera) Dense(1) mse o mae mae, mse
Regresión (solo positivos) Dense(1, activation='relu') mse o msle mae, mape
