Programación de Redes Neuronales Profundas con Keras

Introducción

En esta unidad profundizamos en los conceptos fundamentales de las redes neuronales profundas (DNN - Deep Neural Networks) desde una perspectiva práctica y orientada a la programación en Python utilizando Keras. Keras es la API de alto nivel de la plataforma TensorFlow que proporciona una interfaz accesible y altamente productiva para resolver problemas de aprendizaje automático, con un enfoque en el deep learning moderno.

Nota importante sobre versiones

A partir de TensorFlow 2.16, al instalar TensorFlow con pip install tensorflow se instala Keras 3. Cuando tienes TensorFlow >= 2.16 y Keras 3, por defecto from tensorflow import keras (tf.keras) será Keras 3. Keras 3 es una reescritura completa de Keras que permite ejecutar flujos de trabajo sobre JAX, TensorFlow, PyTorch u OpenVINTO (solo para inferencia), y que desbloquea nuevas capacidades de entrenamiento y despliegue de modelos a gran escala.

Con su enfoque multi-backend, Keras te da la libertad de trabajar con JAX, TensorFlow y PyTorch. Puedes construir modelos que se muevan sin problemas entre estos frameworks y aprovechen las fortalezas de cada ecosistema.

Si necesitas mantener compatibilidad con código antiguo de Keras 2, el paquete legacy Keras 2 sigue publicándose regularmente en PyPI como tf_keras. Para usarlo, puedes instalarlo con pip install tf_keras y luego importarlo con import tf_keras as keras.
