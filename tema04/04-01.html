Librerías de programación de Aprendizaje Automático con Python

1. Pandas: Tu herramienta para manejar datos

Pandas es una librería de manipulación de datos que facilita enormemente el trabajo de los científicos de datos. Desde tareas tan simples como cargar un archivo CSV hasta operaciones más complejas de preparación y limpieza de datos, Pandas lo hace todo más sencillo.

Para entender cómo funciona Pandas, pensemos en la diferencia entre un vector convencional y un DataFrame de Pandas. Un DataFrame es como una tabla de Excel mejorada: tiene índices tanto para las filas como para las columnas, lo que permite organizar y acceder a los datos de manera muy eficiente. Por defecto, Pandas asigna números como índices si no le indicamos otras etiquetas personalizadas. Puedes crear un DataFrame de varias formas: desde un diccionario de Python o desde un array, y luego asignar los valores de los índices que prefieras.

Con Pandas puedes:

  • Consultar rápidamente cuántos elementos hay en cada dimensión (filas y columnas)
  • Ver las dimensiones totales del conjunto de datos
  • Conocer los nombres o etiquetas de filas y columnas
  • Verificar el tipo de dato que contiene cada columna

La función describe es especialmente útil porque te proporciona automáticamente las estadísticas principales de tus datos: promedio, mínimo, máximo, desviación estándar, y más. También puedes crear representaciones gráficas básicas directamente desde el DataFrame. En ocasiones necesitarás trasponer un DataFrame (intercambiar filas por columnas) usando la función T.

Una necesidad frecuente es convertir datos categóricos (como "sí" o "no") a valores numéricos (como 1 o 0), especialmente útil cuando trabajas con valores binarios. Para esto, puedes usar la función replace que sustituye categorías por valores numéricos. Otras funciones muy prácticas son:

  • startswith: verifica si un texto comienza con ciertos caracteres
  • split: divide un texto en partes
  • iloc: accede a datos por posición numérica

También puedes combinar múltiples DataFrames con concat, añadir nuevas columnas con valores calculados a partir de otras columnas, o eliminar columnas que ya no necesitas.
