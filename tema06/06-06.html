Visualización del entrenamiento con TensorBoard

TensorBoard es una herramienta de visualización integrada en TensorFlow que nos permite monitorear el entrenamiento de nuestros modelos en tiempo real. A continuación, te muestro cómo configurarlo en Google Colab (actualizado para 2025).

Configuración en Google Colab:

Primero, cargamos la extensión de TensorBoard para cuadernos Jupyter:

%load_ext tensorboard

Importamos las librerías necesarias:

import tensorflow as tf
import datetime, os

Ejemplo completo con el dataset Fashion-MNIST:

  1. Descargamos el dataset Fashion-MNIST (imágenes de ropa y accesorios)
  2. Normalizamos los datos (los escalamos para que estén en un rango adecuado)
  3. Creamos una red neuronal profunda que incluye una capa de dropout
  4. Al configurar el entrenamiento con model.compile(), añadimos un callback (función de retorno) que conecta con TensorBoard
  5. Iniciamos la visualización de TensorBoard usando su función mágica para notebooks

Ventajas de TensorBoard:

La interfaz de TensorBoard nos muestra:

  • Gráficos interactivos de cómo evoluciona el error y la precisión
  • Visualizaciones de la arquitectura de la red
  • Histogramas de los pesos y sus actualizaciones
  • Y mucho más

Si iniciamos TensorBoard antes del entrenamiento, podemos monitorizar el proceso mientras ocurre, viendo en tiempo real cómo mejora (o empeora) el modelo. En 2025, la API especial para notebooks (tensorboard.notebook) ofrece funcionalidades mejoradas y mejor integración con Google Colab y Jupyter.
